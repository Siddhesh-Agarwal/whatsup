## Execution

**Step 1**: Clone the repo
  ```
  git clone https://github.com/Siddhesh-Agarwal/whatsup
  ```

**Step 2**: Download Dependencies
  ```
  cd whatsup
  npm i
  ```

**Step 3**: Start Development Server
  ```
  npm run dev
  ```

## Tech Stack

1. Next.js 14
2. TailwindCSS
3. TypeScript 5
4. Firebase
    - Firestore Real-time DataBase
    - Authentication
